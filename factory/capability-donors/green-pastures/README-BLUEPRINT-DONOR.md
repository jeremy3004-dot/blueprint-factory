## Copy Policy

This snapshot intentionally excludes local-only or heavy folders:

- `.git`
- `.env.local`
- `.vercel`
- `.artifacts`
- `.playwright-artifacts`
- `.gstack`
- `supabase/.temp`
- `tmp`
- `tsconfig.tsbuildinfo`
- dependency/build folders
- the unrelated large `TEXT with Doctor` folder

Keep this folder as source/reference material. Do not import runtime code directly across site boundaries unless the target site is being intentionally generated from this donor.
