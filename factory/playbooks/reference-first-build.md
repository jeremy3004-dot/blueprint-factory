# Reference-First Build

Blueprint Factory sites start from proven excellence, not from blank-page invention.

## Steps

1. Research at least three sector references.
2. Choose one primary 10/10 donor structure.
3. Capture donor desktop and mobile screenshots.
4. Map donor topology: hero, conversion path, content sections, motion and interaction model.
5. Extract the moves to borrow: layout rhythm, type hierarchy, asset strategy, scroll behavior, CTA placement, and section contrast.
6. Write those moves into `art-direction.md`.
7. Build the client site by translating the donor structure into the client brand, assets, and goals.
8. During Beauty Pass, compare the build to the donor screenshots before assigning scores.

## Rules

- Clone structure and craft, not protected brand identity.
- Prefer sector donors first. If the sector has weak sites, use an adjacent luxury/app/commerce donor and document why.
- If no donor screenshots exist, visual review stays `NOT_READY`.
