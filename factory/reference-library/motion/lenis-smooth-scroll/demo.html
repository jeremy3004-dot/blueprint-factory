Scroll with your wheel/trackpad or click the anchor links.

About this demo: Lenis is an npm library (import Lenis from "lenis") and can't be installed in a double-click, buildless page. So this demo loads Lenis from a CDN <script> if it's reachable; the boot module in lenis-smooth-scroll.js picks it up from window.Lenis. If the CDN is blocked (offline), the module no-ops and you fall back to the CSS scroll-behavior: smooth in lenis-smooth-scroll.css. Either way, under OS "Reduce Motion" the module skips Lenis entirely and the CSS forces instant scrolling.
