# Brief: Blueprint Factory — Live Demos

Status: approved

## What This Site Is

A public Blueprint Factory portfolio page that showcases completed interactive website demos. Visitors browse a horizontal project rail, open any demo as a full independent site in a new tab, and read a static index of every published project.

## Audience

Prospective clients, collaborators, and anyone Jeremy shares the Blueprint Factory portfolio with.

## Required Experience

- Editorial introduction with project count.
- Oversized horizontal carousel with keyboard, button, and touch navigation.
- Visible position indicator.
- Static project index beneath the carousel.
- Live Demo badge and View Live Demo CTA on every card.
- Footer clarifying concept-demo status.

## Deploy Expectation

One stable public Vercel URL for the showcase. Target alias: `blueprint-factory-live-demos.vercel.app`.

## Ready Criteria

- Registry-driven carousel and index stay in sync.
- Lint, typecheck, tests, and production build pass.
- Desktop and mobile screenshots captured for QA.
- Demo URLs verified before public release.
