# Topology: Blueprint Factory — Live Demos

## Route

- `/` — single-page showcase

## Sections (top to bottom)

1. Site header — Blueprint Factory mark + Live Demos subtitle
2. Introduction — headline, copy, project count stats
3. Horizontal demo carousel — one large card per published project
4. Static project index — full list with thumbnails and CTAs
5. Footer disclaimer — concept demonstration language

## Interaction Model

- Carousel uses native horizontal scroll with scroll snapping.
- Desktop previous/next buttons scroll the rail.
- Keyboard left/right arrows move between cards when focused.
- Touch swipe works on mobile without custom drag physics.
- No auto-advance; visitors control pace.
- `prefers-reduced-motion` disables smooth scrolling transitions.

## External Links

Each card and index row opens its demo in a new browser tab via **View Live Demo**.
