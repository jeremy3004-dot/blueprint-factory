# Brief: one-and-only-resorts

Status: draft

## What This Site Is

A reference-first clone of the One&Only Resorts homepage, built as a premium hospitality website inside Blueprint Factory.

## Audience

Luxury travelers and reviewers evaluating whether the clone captures One&Only's calm resort editorial rhythm, mobile polish, and cinematic arrival.

## Inputs Provided

- User request: clone `https://www.oneandonlyresorts.com/?utm_source=chatgpt.com`.
