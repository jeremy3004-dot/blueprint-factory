# Accommodation Page Specification

- Target: `src/app/accommodation-in-pokhara/page.tsx`.
- Interaction model: static editorial stories with one-shot reveal and hover-gated image scale.
- Content: Dorje Suite, Jen Suite, Deluxe Room and Standard Room. Descriptions and amenities must come from the current Dorje's page. Do not publish rates until the client resolves conflicts between the overview and current detail pages; use a “Check availability” action.
- Assets: four client room image groups, locally downloaded and logged.
- Responsive: contained 4:3 hero on desktop, edge-to-edge mobile; alternating media/text becomes a single stacked reading order.
