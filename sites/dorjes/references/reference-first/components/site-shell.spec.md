# Site Shell Specification

- Target: shared header, menu, footer and booking close components.
- Interaction model: click-driven menu; sticky header; keyboard Escape and focus return.
- Donor evidence: compact pale masthead, centered wordmark, quiet utility controls and dense footer in `donor-1440.png` and `donor-390.png`.
- Production identity: Dorje's logo, parchment background, ink/brass palette, Stay/Wellness/Resort/About/Contact navigation and external Book action.
- Responsive: desktop utility row becomes a compact menu/logo/book row; footer moves from four columns to two and finally one where needed.
- Accessibility: semantic nav, labelled menu control, focus trap or logically contained focus, visible focus rings, no inaccessible icon-only actions.
